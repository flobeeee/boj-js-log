## 풀이 아이디어

큰 숫자대로 정렬해서 목표값에서 - 하기 

## 코드

```js
const input = require('fs').readFileSync('/dev/stdin', 'utf8').trim().split('\n');
const [n, k] = input[0].split(' ').map(Number);
const coins = input.slice(1, n + 1).map(Number);

let count = 0
let goalMoney = k

for (let i = n-1; i >= 0; i -- ) {
  if (coins[i] <= goalMoney) {
    const remain = Math.floor(goalMoney / coins[i]) // 몫 구하기

    goalMoney = goalMoney - (coins[i] * remain)
    // goalMoney = goalMoney % coins[i] // 이 연산자도 사용 가능
    count = count + remain
  }
}

console.log(count)

```

## 시간복잡도

- 내 풀이: O(n)
- 이유: 동전 종류 수 n번만 순회
